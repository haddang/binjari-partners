FAQ

자주 묻는 질문

01 스마트 콘센트 설치 과정이 복잡한가요?
기존 벽면 콘센트에 스마트 플러그를 꽂기만 하면 무선 대시보드에 자동 등록되는 플러그 앤 플레이 방식입니다. 도입 시 전담 매니저가 매장을 직접 방문하여 상세 가이드를 제공합니다.
02 월 이용료와 기기 대여 비용은 어떻게 산정되나요?
매장 좌석 규모에 따라 요금제가 세분화됩니다. 첫 30일은 서비스 무료 체험이 가능하며, 구독 시 대기전력 자동 제어 콘센트 대여 비용이 포함됩니다.
03 기존에 사용 중인 무인 키오스크와 통합 연동이 가능한가요?
주요 무인 키오스크 제품들과 API 및 하드웨어 연동을 통한 실시간 동기화를 지원합니다. 추가 조율이 필요한 경우 당사 지원 부서에서 신속히 연동을 도와드립니다.
04 인터넷이나 Wi-Fi가 일시적으로 끊기면 제어가 불가능해지나요?
인터넷이 끊겨도 스마트 게이트웨이의 자체 로컬 메모리를 기반으로 정상 작동합니다. 통신 복구 즉시 미전송 전력 데이터가 대시보드로 자동 업로드됩니다.

Fast Helpline

더 빠른 상담이 필요하신가요?

글로 문의를 작성하기 번거로우시다면 실시간 카카오톡 채널 상담을 이용해 보세요.

실시간 카카오톡 채널 상담

카카오톡에서 '빈자리 파트너즈'를 검색해 추가하시면 매니저와 1:1 채팅을 통해 막힘없이 실시간 문답을 나누실 수 있습니다.

대표 전화 및 기술 지원

스마트 기기 사용법이나 무선 연결 장애로 영업에 지장이 생기신 경우, 대표 전화(1588-3456)로 전화주시면 바로 친절하게 안내해 드립니다.

매장의 빈자리를 채우는 가장 스마트한 방법. 실시간 관제, 통합 예약, 데이터 인사이트.
